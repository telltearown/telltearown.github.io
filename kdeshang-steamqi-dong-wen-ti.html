KDE上Steam启动问题

October 8, 2025 是饕餮哇
Linux Steam

当你使用X11或者是Wayland的图形界面程式的话，你会发现Steam只要装了Deb包在你的系统上的那一刻图形化打开方式已经爆炸了。然而这个时候你还想着终端启动它那就太难了。于是反复逼问AI的情况下，Linux小白已经完全掌握了！（助威！！！）

首先把当前环境全部导出到文件：

env | grep -E '^(PATH|LD_|GTK_|QT_|XDG_|GALLIUM|VK_|DRI|LIBGL)' >~/.steam_env

再给Steam单独写一行：

echo 'export GDK_BACKEND=x11' >> ~/.steam_env

建一个 “带环境” 的包装脚本：

mkdir -p ~/.local/bin
nano ~/.local/bin/steam-wrapper

加入这些：

#!/bin/bash
# 先加载刚才dump下来的完整环境
set -a
source ~/.steam_env
set +a
# 再启动真正的steam
exec /usr/bin/steam "$@"

赋予权限：

chmod +x ~/.local/bin/steam-wrapper

复制一份 desktop 文件到本地：

cp /usr/share/applications/steam.desktop ~/.local/share/applications/
sed -i 's|Exec=/usr/bin/steam|Exec=/home/'$USER'/.local/bin/steam-wrapper|' \
       ~/.local/share/applications/steam.desktop
